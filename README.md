# Frontend Builder

A flexible frontend development platform supporting React, Vue, Angular, HTML, CSS, and SCSS.

## 🚀 Features
- Multi-framework support
- Modular architecture
- Local development environment

## 📦 Planned Stack
- Vite (build tool)
- React / Vue / Angular support
- SCSS support
- ESLint + Prettier

## 🛠 Getting Started
```bash
npm install
npm run dev
```
